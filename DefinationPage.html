Common Web Terms

A short definition list using <dl>, <dt>, and <dd> — semantic and accessible.

HTML
HyperText Markup Language — the standard markup language used to create web pages and web applications. It structures content (headings, paragraphs, lists, links, images).
CSS
Cascading Style Sheets — the style sheet language used to describe the presentation (layout, colors etc) of HTML content.
JS
JavaScript — the programming language used to add interactivity and behaviour to web pages.
HTTP
Hypertext Transfer Protocol = the protocol used for transferring web pages and data between client and server.
